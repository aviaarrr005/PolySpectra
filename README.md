## How to Run This Project

1.  Clone the repository.
2.  `py -3.13 -m venv .venv`
3.  `.\.venv\Scripts\activate`
4.  `pip install -r requirements.txt`
5.  `flask run`
6.  Open `http://127.0.0.1:5000` in your browser.
